# AI烧烤店SaaS — 智能餐饮管理系统

> 🔥 面向中小餐饮门店的一站式智能管理系统，以 **AI 自然语言点餐** 为核心交互范式。

## 核心特性

- 🤖 **AI智能点餐**：基于 DeepSeek Function Calling，自然语言对话即可下单
- 📋 **扫码点单**：分类浏览 + 口味选择 + 购物车
- 🍳 **KDS厨房屏**：按桌/按菜双视图，实时菜品状态流转
- 💰 **收银结账**：订单汇总 + 一键结账
- 📦 **库存管理**：实时扣减 + 低库存告警 + 一键补货
- 📊 **数据报表**：营收分析、热销排行、客单价趋势、顾客画像（RFM）
- ⚙️ **后台管理**：菜品CRUD、桌台分区、员工管理

## 快速开始

1. 克隆本项目
2. 用浏览器打开 `output/AI烧烤店SaaS_Demo.html`
3. 如需启用AI点餐，将文件中的 `sk-YOUR_DEEPSEEK_API_KEY` 替换为你的 DeepSeek API Key

## 技术栈

HTML / CSS / JavaScript / DeepSeek Function Calling API
